# Android动态换肤（一、内置皮肤）


本地内置多套皮肤的资源文件，使用 SharedPreferences记录当前设置的皮肤序号，当选中某一套皮肤后，使用SkinUtil类将这套皮肤的资源ID获取到，然后再每个activity中设置皮肤。
